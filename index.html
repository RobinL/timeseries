Time series simulator

Help

'Bigerror': specify a large error which will be inserted into the series. The visualisation will then show how this error propagates through the series. Set input box to empty if you don't want a big error

'Betas/Gammas': A single value is the parameter on the first time lag. Write a comma separated list of values for longer lags. E.g. if betas = 0.8,0.5 this is saying that b1 = 0.8 and b2 = 0.5

Forecast: The line in green is the best forecast possible given the data generating mechanism

Tips

Don't put numSims to more than about 500 if you don't want the page to hang

Higher values of numSims require lower values for opacity
